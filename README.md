# FastAPI Auth Service

Production-minded authentication backend skeleton (no auth endpoints yet).

## Quick Start

1. Copy `.env.example` to `.env` and fill in values.
2. Start database with `docker-compose up -d db`.
3. Run the app with `uvicorn app.main:app --reload`.
4. Run tests with `pytest`.

## Structure

See `app/` for application code and `tests/` for test scaffolding.
